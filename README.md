# Webhook

Simple API service to executes shell script inside host directory

## Running Services

Build
```bash
./_scripts/build.sh
```

Create and edit config file
```bash
cp example.yaml _bin/webhook.yaml
```

Run service
```bash
./_scripts/run.sh
```
